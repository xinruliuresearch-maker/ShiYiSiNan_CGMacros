# Nature Food CGM Digital Phenotype Package

Date: 2026-06-20

This package archives the integrated Nature Food / CGM digital phenotype work
developed in `D:\ai for science`. It connects NHANES food-system plasticizer
exposure anchoring, CGMacros meal-matrix PPGR vulnerability, and JAEB/Loop/PSO
CGM phase-map validation into a single publication-oriented research package.

## Main Entry Points

- `results/nature_food_p0_publication_lift_2026_06_18/preview/index.html`:
  rendered preview entry for the Nature Food package.
- `results/nature_food_p0_publication_lift_2026_06_18/docs/NatureFood_next_tasks_to_publication_zh_2026_06_18.md`:
  next task design toward Nature Food.
- `results/nature_food_p0_publication_lift_2026_06_18/docs/NatureFood_presubmission_enquiry_v1.0.md`:
  presubmission enquiry draft.
- `results/nature_food_p0_publication_lift_2026_06_18/preview/nature_food_main_figures.html`:
  rendered six-figure preview.
- `results/nature_food_p0_publication_lift_2026_06_18/software_release/integrated_food_cgm_phenotype_v0_1/`:
  integrated phenotype reproducibility prototype.

## Included

- `scripts/`: analysis and packaging scripts from the integrated workstream.
- `docs/`: planning, statistical, manuscript and cross-project design notes.
- `figures/`: CGM phase-map and journal-grade diagnostic figures.
- `results/`: Nature Food, CGMacros, NHANES, JAEB and npj Digital Medicine
  derived result packages.
- `package_file_manifest.csv`: file manifest for this package.
- `data_exclusion_manifest.csv`: raw/external data inventory excluded from this
  Git snapshot.

## Excluded Raw Data

The source workspace contains approximately 4 GB of raw or external datasets,
including JAEB public zip files. These are intentionally excluded from this
snapshot to avoid pushing large third-party data archives into Git. The excluded
files are listed in `data_exclusion_manifest.csv`.
